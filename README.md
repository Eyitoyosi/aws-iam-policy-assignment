# AWS IAM Policy Assignment

## Objective

This repository contains IAM policies for:

- S3 access control
- IAM user management
- IAM group management

## Assignment 1

Teacher policy:
- Upload files
- Download files
- No delete access

Student policy:
- Download only

Administrator policy:
- Full access

## Assignment 2

Junior admin policy:
- Create users
- View users
- Reset passwords
- No delete access
- No access key creation

## Assignment 3

Team lead policy:
- Create groups
- Add/remove users
- View groups
- Cannot delete groups
- Cannot attach AdministratorAccess policy
